# Installation Package

Source is available as source-package on NPM. To install the package :
<Tabs items={['pnpm', 'yarn']}>
  <Tabs.Tab>
    ```shell
    npm install source-package
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```shell
    yarn install source-package
    ```
  </Tabs.Tab>
</Tabs>
